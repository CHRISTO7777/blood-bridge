🩸 BloodBridge

BloodBridge is a full-stack web application designed to connect blood donors with those in need. It provides a seamless platform for users to register as donors, making it easier to manage and access donor information in real-time.

🚀 Features
 Donor Registration with essential details (Name, Blood Group, Location, Contact)
 Real-time data storage using MySQL
 Fast and responsive frontend built with React + Vite
 Backend API powered by Node.js and Express
 Simple and clean UI for easy interaction



🛠️ Tech Stack
Frontend: React, Vite, Tailwind CSS
Backend: Node.js, Express
Database: MySQL
API Communication: REST (Fetch API)
